# TFT Set 17 对比型数值计算器 MVP

这是一个无依赖的前端 MVP，用来验证下面这件事是否可行：

- 选定一个英雄
- 在固定目标和固定时间窗口下
- 对比不同 `装备 / 奥术装备 / 光明装备 / 纹章 / 羁绊 / 强化符文/Buff`
- 输出和参考站相近的表格
- 勾选后生成累计伤害曲线、事件日志和伤害分布图

## 当前实现

- 顶部分类切换
- 英雄选择、星级选择、目标预设选择
- 表格列：
  - `Extra`
  - `DPS at 5`
  - `10秒输出`
  - `DPS at 15`
  - `20秒输出`
  - `25秒输出`
  - `额外输出(25s)`
  - `加入对比`
- 勾选右侧复选框后生成：
  - 累计伤害折线图
  - Index Log 事件日志表
  - Damage Distribution 环形图

## 当前数据状态

当前版本使用的是本地示例数据，主要目的是打通：

- 页面结构
- 计算流程
- 表格和图表联动
- 分类切换
- 对比勾选逻辑

这意味着：

- UI 与交互已经可验证
- 计算引擎与图表输出结构已经可验证
- 真实 Set 17 数据仍需要后续接入抓取器或手工结构化数据

## 运行方式

建议用任意本地静态服务器打开本目录，例如：

```powershell
python -m http.server 8000
```

然后访问：

```text
http://localhost:8000
```
